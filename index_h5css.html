    1. css means cascading style sheet.
    2. css can be applied to web pages in three approaches - inline, internal, external
    3. h1 to h6 tag is used for headings on web pages. h1 is biggest size of text and h6 is lowest size of heading text..

    7. For internal CSS (applied on h2 tag), add style tag in head section of web page and define the element styling definition.
    8. The css element definition's lines are separated by semicolon
    9. CSS selectors : Four types - Id selectors, Class selector, Universal selector, Group selector
    10. for ID selector, the CSS definition name would have hash(#), id attribute of element and class selectors we use dot(.) and class attribute of element. For Universal selector astrisk * is used in definition.
    11. ID selector CSS would be used for applying style to a specific element of web page not to all elements/tags. E.g. above line of para1

    ... meta tags of web pages..

    1. meta tag means data about data.
    2. meta name equals viewport means responsive web page to all device types.
    3. meta name equals keywords content html kya hai make the page ranked by google for this string
    4. meta name equals description what is java string value will work with above keywords string value will work for ranking of web page.
    5. meta name equals robots , contents follow index means page will be indexed and followed by bots of search engine.
    6. meta name equals robots , contents nofollow noindex means page will be NOT indexed and NOT followed by bots of search engine.
